Ubuntu 常用指令

发表于2019-05-02|Ubuntu
|字数总计:135|阅读时长:1分钟|阅读量:

前言

最近在Ubuntu系统下搭了一个游戏服务器，简单总结一下一些基本指令

基本指令

  • mkdir 指令

    建立一个文件夹

    mkdir root

    即在当前路径下建立root文件夹

  • cd 指令

    进入文件夹

    cd root

    在当前路径下进入root文件夹；当然也可以使用绝对路径

  • wget指令

    wget http://xxxxxxx.xx

    从http地址下载文件，详细自查百度

  • touch指令

    touch xxx.sh

    建立文件指令；上述指令建立一个脚本
